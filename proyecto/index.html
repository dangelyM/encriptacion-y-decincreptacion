Codificador y Decodificador

Bienvenidos a un sintema de cifrado, este sistema te permite cifrar y descifrar texto a partir de un algoritmo de cifrado. Para cifrar un texto simplemente hay que poner en el recuadro superior el texto que quieres cifrar y poner una contraseña. Después de enviarlo desaparecerá el texto original del recuadro y aparecerá el texto cifrado. para decifrar un texto colocas el texto cifrado luego de reenviarlo te aparecera el texto original. .
